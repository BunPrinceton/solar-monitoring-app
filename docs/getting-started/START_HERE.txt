KEY RECOMMENDATIONS
================================================================================

INVERTER: Use Local Gateway API (pypowerwall)
  Why: Zero cost, works offline, 5-second updates
  Alternative: Tesla Fleet API (cloud, for Powerwall 3)
  
METER: Use AI-on-the-Edge (ESP32-CAM)
  Why: Affordable ($15-20), non-invasive, 99%+ accuracy
  Alternative: Shelly EM ($50-100, requires electrical work)

API SERVER: Flask + InfluxDB + Grafana
  Why: Proven stack, all open source, self-hosted

INTEGRATION: MQTT for meter, pypowerwall for inverter
  Why: Standard protocols, widely supported, reliable
